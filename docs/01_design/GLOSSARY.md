# Glossary

## What this is
Short definitions for recurring terms in this repository.

## What it's for
Reduces ambiguity when humans or agents review the app or docs.

## How to create or update
Add a term here when it appears in code or docs more than once.

## Terms
- **Overview** — the landing page that introduces the profile and highlights content.
- **Open Source** — the repository-style projects section.
- **Blog** — the article list and article detail experience.
- **Readme** — the in-app README-style profile page.
- **SSR** — server-side rendering.
- **Prerender** — static generation of selected routes at build time.
- **Signals** — Angular reactive state primitives.
- **Standalone component** — an Angular component that imports its own dependencies.
- **YouTube embed** — the embedded video section on article pages.
